# L'AVENIR Furniture Store
A modern and elegant furniture store website designed with a focus on luxury and minimalism.

## Features
- Clean, responsive design using Flexbox.
- Premium typography with Playfair Display.
- Interactive product gallery.
- Professional Call to Action (CTA) section.

## Tech Stack
- HTML5
- CSS3 (Flexbox & Grid)
- Google Fonts & Font Awesome
